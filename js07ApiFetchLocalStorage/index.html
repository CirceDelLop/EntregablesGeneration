JS 07 API Fetch y Local Storage

Sesión Local Storage

El Local Storage es una API de almacenamiento del navegador que permite guardar datos de manera persistente en el cliente (navegador) sin fecha de expiración. Esto significa que los datos permanecen incluso después de cerrar el navegador, a menos que el usuario los borre manualmente.

Métodos principales

  • localStorage.setItem('clave', 'valor'): Guarda un valor.
  • localStorage.getItem('clave'): Recupera un valor.
  • localStorage.removeItem('clave'): Elimina un ítem específico.
  • localStorage.clear(): Elimina todos los ítems guardados.
